# The Commons v0.1

A public platform for the people.

## Philosophy
- Worth precedes engagement
- Power to the People
- No advertising. No data selling. No biometrics. No AI buttons.
- Truth protected. Misinformation never published.
- Local small businesses and creators only.
- Flat $1 transaction fee. Operating costs only. No profit.

## Founding Authority
Sovereign Human T.L. Powers

## Stack
- Python + FastAPI
- SQLite (development) → PostgreSQL (production)
- Jinja2 templates
- Plain HTML/CSS/JS — no frameworks, no clutter

## Modules
- **config.py** — settings and environment
- **database.py** — database connection and setup
- **models.py** — all data models
- **auth.py** — user registration and login (no biometrics)
- **posts.py** — content creation and feed
- **circle.py** — governance and Circle management
- **fingerprint.py** — truth verification system (invisible to users)
- **commerce.py** — marketplace and $1 transaction system
- **codex.py** — The Commons Codex embedded in code

## Run

    pip install -r requirements.txt
    python main.py

## Development
    
    python main.py --dev     # Development mode
    python main.py --check   # Check configuration
